Game Objective:

Two players are needed to play, select which player you wish to be and use the key that is assigned to that player. Once the game has started you need to press your key to advance your car.
Please choose which player you want to be and select I'm ready! Once both players are ready select the start game button and spam your key!

To Win the Game:

The first player to make it to the end of the track wins!
